# C# Basic Examples

This project contains various C# examples covering basic to intermediate topics.

## Installation & Setup

If you have downloaded this project, follow these steps to get started:

1. **Prerequisites**: Ensure you have the [.NET SDK](https://dotnet.microsoft.com/download) installed.
2. **Open Terminal**: Navigate to the project folder where `cSharpBasic.csproj` is located.
   ```powershell
   cd cSharpBasic
   ```
3. **Build the Project**: Run the following command to restore dependencies and build the application.
   ```powershell
   dotnet build
   ```

## How to Run

To run a specific topic, use the `dotnet run` command followed by the topic name.

```powershell
dotnet run [TopicName]
```

### Examples:

- **Variables**: `dotnet run VariableTypes`
- **Control Flow**: `dotnet run ControlFlow`
- **Functions**: `dotnet run Functions`
- **Arrays**: `dotnet run Arrays`
- **Exception Handling**: `dotnet run ExceptionHandling`
- **Generics**: `dotnet run Generics`
- **Delegates**: `dotnet run Delegates`
- **Lambda Expressions**: `dotnet run LambdaExpression`
- **Events**: `dotnet run Events`
- **String Operations**: `dotnet run StringOperations`
- **LINQ**: `dotnet run LinqExample`
- **File I/O**: `dotnet run FileIo`
- **Tuples**: `dotnet run Tuples`
- **Nullable Types**: `dotnet run NullableTypes`

**Note:** The arguments are case-insensitive (e.g., `dotnet run variabletypes` works too).
